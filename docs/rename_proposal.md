# プロジェクト名変更案

現在のプロジェクト名: **tazuna**
現在の由来: 「手綱」を握ってコントロールするイメージ。

## 新コンセプト
- **Lazy (怠惰)**: 開発者は極力働かない。
- **コーディングの完全自動運転**: 人間の介入を最小限にする自律型エージェント。

「手綱を握る」必要すらなく、完全に任せてしまえるものを目指すためのリネーミング案です。

## 提案候補

### 1. Sloth (スロース)
- **意味**: ナマケモノ、怠惰
- **コンセプト**: 「怠惰」を象徴する動物。ユーザーはナマケモノのように動かず、エージェントが全てを行う。
- **メリット**: キャッチーでロゴなどのキャラクター展開がしやすい。「Lazy」そのもの。
- **イメージ**: 木にぶら下がっている間に開発が終わっている。

### 2. Passenger (パッセンジャー)
- **意味**: 乗客
- **コンセプト**: 自動運転車において、人間はドライバーではなく「乗客」になる。
- **メリット**: 「完全自動運転」という思想に最も合致する。ユーザーは目的地（要件）を伝えるだけで、あとは座っているだけ。

### 3. Snooze (スヌーズ)
- **意味**: （目覚ましを止めて）うたた寝する
- **コンセプト**: 「あと少し寝ている間にやっておいて」という究極のLazyさ。
- **メリット**: リラックスした雰囲気。夜中にエージェントが働いてくれるイメージ。

### 4. LazyPilot (レイジーパイロット)
- **意味**: 怠惰なパイロット / 自動操縦
- **コンセプト**: Copilot（副操縦士）ではなく、主導権を持つが「Lazy」なパイロット、あるいはユーザーがLazyでいられるためのパイロット。
- **メリット**: 既存の "Copilot" 系の名称に対するアンチテーゼ的な響き。

### 5. Limo (リモ)
- **意味**: リムジン
- **コンセプト**: 運転手付きの高級車。快適に後ろ部座席に座っているだけで目的地に着く。
- **メリット**: 高品質・VIP待遇な体験を想起させる。

## 所感
「Lazy」を前面に出すなら **Sloth** が最も象徴的です。「完全自動運転」の機能的側面を強調するなら **Passenger** が適しています。
tazuna（手綱）からの脱却という意味では、自ら動かない **Sloth** が思想をよく反映していると考えます。
