## [0.1.0] - 2024-01-01

### Added
- Initial release of dursor
- Multi-model parallel execution (OpenAI, Anthropic, Google)
- BYO API key support with encrypted storage
- GitHub PR creation and update functionality
- Visual diff viewer for comparing model outputs
- Conversation-driven development workflow
- Self-hostable deployment with Docker support
- Claude Code, Codex, and Gemini CLI executor support
- GitHub App authentication

### Technical
- FastAPI backend with async support
- Next.js 15 frontend with React 19
- SQLite database with aiosqlite
- Fernet (AES-128) encryption for API keys
